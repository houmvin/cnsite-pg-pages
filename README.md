# cnsite-pg-pages

本仓库用于归档和发布多个独立的 HTML 页面。页面内容涉及不同主题，各自独立，不针对任何单一域名或具体网站。

## 项目简介

这是一个静态页面归档仓库，主要存放各类 HTML 文档。每个页面均为独立文件，可直接通过浏览器打开查看。仓库本身不提供交互式服务、数据库或后端逻辑，仅作为页面文件的存储与分发点。

## 目录说明

- `pages/`：存放所有 HTML 页面文件。每个文件命名清晰，便于区分。
- `assets/`（如有）：存放页面所需的样式、脚本、图片等静态资源。
- `README.md`：本说明文件。

## 页面归档说明

归档的页面均为历史或功能性 HTML 文档，可能包含以下类型：
- 信息展示页面
- 简单工具界面
- 静态内容存档

每个页面独立存在，不相互依赖。页面内容可能随时间更新或移除，恕不另行通知。
